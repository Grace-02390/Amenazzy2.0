Notas de la Aplicación Topitop
Descripción

Topitop App centraliza catálogo de productos, gestión de inventarios, programa de fidelización y compras. Está diseñada para ser escalable y modular.

Características principales

Visualización de catálogo y disponibilidad de productos.

Carrito y proceso de compra con múltiples medios de pago.

Seguimiento de pedidos y notificaciones personalizadas.

Gestión de inventarios y control interno de stock.

Tecnologías principales

Frontend: Flutter, Dart, Firebase.

Backend: Node.js, Express, PostgreSQL, Redis.

Infraestructura: Google Cloud, Docker, CI/CD con GitHub Actions.

Instalación rápida

Clonar repositorio:

git clone https://github.com/topitop/app.git
cd app


Configurar .env con credenciales internas.

Instalar dependencias:

flutter pub get


Ejecutar en desarrollo:

flutter run

Testing
flutter test
